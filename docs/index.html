Professional self-hosted Discord music bot with web dashboard, unlimited queues, and custom playlists. The open-source alternative to discontinued public bots.

Why Choose Bassline-Bot?

The most advanced self-hosted Discord music bot with enterprise-grade features

🌐

Web Dashboard

Beautiful real-time dashboard to monitor and control your bot from anywhere. View statistics, manage queues, and configure settings through a modern web interface.

🎵

High-Quality Audio

Professional FFmpeg audio processing with bass boost, volume control, and crystal-clear streaming from YouTube with intelligent format selection.

📋

Custom Playlists

Create server-specific playlists using Discord channels. Collaborative playlist building where your community can add songs together.

🔧

Admin Controls

Comprehensive admin panel with DJ roles, permission management, usage analytics, and customizable server settings for complete control.

🐳

Docker Ready

Production-ready Docker deployment with docker-compose. Includes monitoring, health checks, and automatic restarts for 24/7 reliability.

📊

Monitoring & Metrics

Prometheus metrics, health checks, usage analytics, and performance monitoring. Track everything from command usage to system resources.

Quick Setup Guide

Get your Discord music bot running in under 10 minutes

1

Clone Repository

git clone https://github.com/Cthede11/bassline-bot.git cd bassline-bot
2

Configure Bot

cp .env.example .env # Add your Discord bot token
3

Start Bot

docker-compose up -d # Dashboard: localhost:8080
